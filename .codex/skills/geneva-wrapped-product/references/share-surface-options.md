# Share Surface Options

Choose one main surface. The others should support it.

## Option 1: Fifa Wrapped As Primary

Best when you want the richest current data and the closest thing to a full recap.
This is the default recommendation for a production-grade Geneva wrapped.

## Option 2: Walk-In As Primary

Best when the main task is story-system design, beat ordering, or share-card planning.
This is strongest as the spec and experimentation surface, not automatically the final production route.

## Option 3: Wrapped Family As Primary

Best when the goal is one memorable scene or a spend-focused flex.
This is not yet the strongest choice for a broad multi-beat recap.

## Recommended Pattern

- `Primary engine`: Fifa Wrapped
- `Story lab`: Walk-In
- `Special scene`: Wrapped Family

Use a different pattern only if product goals clearly demand it.
