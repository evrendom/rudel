# Candidate Beats

These are the strongest Geneva beats with current data.

## Highest-Confidence Beats

- `Top skill`
  Strong identity signal and already tracked.

- `Commit rate plus success rate`
  Good proof-of-work card. Stronger when framed as delivery or execution quality.

- `Top project`
  Concrete and easy to picture. Better than another abstract usage metric.

- `Top slash command or subagent`
  Reveals workflow extension and structure.

- `Plan mode rate`
  Good habit card for power users.

- `Dominant archetype`
  Works if clearly framed as inferred behavior, not a deeply personal truth.

- `Peak day or streak`
  Strong remarkable-moment card.

- `Model relationship`
  Good only if framed as preference, rotation, or evolution rather than fake loyalty.

## Medium-Confidence Beats

- `Primary error pattern`
  Good for advanced audiences. Riskier for broad celebratory runs.

- `Token shape`
  Useful for power users, but not naturally emotional.

- `Repository breadth`
  Better as supporting evidence than as a hero beat.

## Avoid Until Data Changes

- favorite local time or favorite day
- average context window percentage
- a durable user archetype share card that implies persisted identity state

## Selection Rule

Pick beats that combine:
- self-recognition
- public meaning
- visual clarity
- defensibility

Do not pick a beat only because the metric exists.
